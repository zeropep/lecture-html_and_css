switch case 문은 if - else if - else 문과 비슷한 형태의 조건문

  • switch(값) : true / false 가 아닌 특정 값을 비교선언한다.
  • 각 조건은 case 가능성의 값 : 수행코드 break; 형태로 만들어짐
  • else문과 같은 역할은 default : 수행코드 break;를 사용한다.
  • type을 정확히 판단해서 값을 배정해야 한다.

if문과의 비교

  • if문은 한번 수행하면 나머지 조건은 아예 수행하지 않으므로 가능성이 높은 것을 위로 올리는 것이 수행속도에 영향을 끼친다.
  • 하지만 switch 문은 이러한 조건에서 조금 더 자유롭다.
  • 서로다른 범위에 있는 요소들을 비교하기에 조금 더 좋다.
  • switch문은 일종의 캐시형식으로 작동하므로 수행속도가 더 빠를 가능성이 높다.
