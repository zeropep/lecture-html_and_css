중첩 반복문

  • 반복문 안에 반복문을 사용하는 방식
  • 원리는 바깥쪽 반복문이 1번 수행될 때 안쪽 반복문은 청므 부터 끝까지 수행
  • 구구단을 생각하면 2, 3, 4, ...9단을 출력하는 건 바깥쪽 반복문
  • 각 단에 2~9까지 곱하는건 안쪽 반복문

구구단 출력하기

문자데이터를 분석하고 단어를 조합하여 새로 생성하기

  • 특정 문자열 뒤 혹은 앞에 문자를 추가하여 새로운 문자데이터를 만드는 방식
  • 마케팅이나 광고관련 프로그램에서 많이 사용하며
  • 데이터 분석 분야에서 데이터를 원하는 형식으로 전처리 할 때도 많이 사용됨
