string method & property

  • 문자는 길이 속성을 갖고 있으며 문자열.length로 알 수 있음
  • 길이를 갖고 있다는 것은 시작과 끝이 있다는 개념으로 이해
  • 그러므로 순서가 존재하고 그 순서를 숫자화 시킨 index라는 것이 존재
  • 문자의 값은 불변(immutable)의 속성으로, 인덱스를 활용해 값 자체를 변화시킬 수 있음
  • 인덱스는 0부터 시작하기 때문에 마지막 인덱스는 length -1이라는 공식이 성립한다.
  • 인덱스를 활용한 다양한 method가 있으며, 실제 실무에서 문자관련 method가 많이 사용됨
  • 띄어쓰기도 포함됨
