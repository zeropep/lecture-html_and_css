Java Script Data Type

  • Number : 숫자 - 정수, 실수, NaN
  • String : 문자 - quotes로 감싸진 모든 값
  • Boolean : 참/거짓 - true / false
  • Object : 객체
  • Array : 배열(객체)
  • undefined : 미정된 값
  • Null : 알수없음

typeof : data type을 알고 싶을 때 사용하는 키워드

자바스크립트의 기본 데이터 타입: primative type

  • number
  • boolean
  • string(원래는 참조형 데이터 타입인데 특수하게 변형된 형태)
  • 다른 프로그래밍 언어에서는 string은 참조형 데이터 타입

자바스크립트의 참조 데이터 타입: reference type

  • string
  • object
  • array
  • function etc...
