자바스크립트 언어의 사용

  1. 스크립트 태그 안에 작성한다.
  2. 파일이름.js 를 만들고 코드를 작성한 후 script src="경로"로 해당 파일을 불러온다.
  3. 이외에도 가능은 하나 외부에서 불러오기가 실행이 안될 수 있음

자바스크립트 코드의 위치

  1. head 태그의 안쪽 : 주로 함수 선언식 등의 실행코드가 아닌 호출대상 코드들
  2. body 태그의 안쪽(주로 가장 하단) : 선언식, 표현식(실행) 등의 코드
  3. 왜냐하면 js는 html을 동적으로 실행시켜야 하기 때문에 모든 요소를 읽고 난 후 읽어들이는 것이 좋음.
