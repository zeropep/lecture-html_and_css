DOM : Attribute, Attribute Value 컨트롤

  • HTML 태그의 속성과 속성값에 접근하여 수정, 삭제 가능
  • HTML 태그의 속성을 토글(toggle)할 수도 있다.
  • 속성을 컨트롤 한다는 것은 태그의 특성을 변경하는 것이기 때문에
  • 속성에 관련된 property의 종류와 응용 방법도 다양하다.
  • Element(tag)에 setAttribute(name, value)를 많이 사용하며
  • class나 id에 접근할 때 사용하는 classList, id property도 많이 사용된다.
