Mouse Event

  • click : onclick, dbclick(더블클릭), contextmenu(우클릭)
  • wheel : wheel, scroll, auxclick
  • drag & drop : drag, [dragstart, dragleave, dragover, dragenter, dragend], drop
    • // 괄호 부분은 보통 생략함
  • focus & blur : 폼태그에 커서 이동, 커서 아웃
  • input, change : 입력된 상태, 입력한 값이 바뀌는 상태
  • mouseover, mouseleave 등 : 마우스 포인터의 위치값에 반응
  • submit : 폼태그가 제출될 때
