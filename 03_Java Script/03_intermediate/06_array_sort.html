배열의 정렬

  • 문자와 숫자의 정렬 방식이 다름
  • 문자는 배열.sort()는 순차정렬, 역정렬은 배열.sort() 한 후 배열.reverse()
  • 숫자는 배열.sort(function(a, b){return a - b} 이면 순차정렬
  • 숫자는 배열.sort(function(a, b){return b- a} 이면 역정렬
  • 유사배열인 객체안의 값을 정렬하는 방법
    • 객체의 property 값을 통해 정렬한다
    • 문자 정렬은 배열의 정렬함수를 활용하여 파라미터에 프로퍼티 접근을 한 후 연산
    • 숫자 정렬도 문자 정렬의 방식과 같음
