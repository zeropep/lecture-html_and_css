Set

  • 동일한 값은 한 번만 저장하는 객체
  • 키의 개념은 있으나 물리적인 키의 구현은 되어있지 않다.
  • value 값이 곧 키의 역할을 하기 때문에 중복을 허용하지 않는 구조
  • length 대신 size라는 property를 사용하여 크기를 알 수 있음.
  • forEach method로 순환 탐색 가능하며 객체타입이므로 instanceof를 사용함
  • 전용 메서드는 add(), delete(), has(), clear(), keys(), values(), entries()
  • 배열을 통해 객체를 생성할 수 있음.
