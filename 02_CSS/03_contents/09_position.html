Position

  • fixed : 브라우저를 기준으로 특정 위치에 고정으로 표시
  • static (default) : 정상적인 흐름
  • relative : 원래 위치로부터 상대적인 요소(element)
  • absolute : 해당 요소를 감싸고 있는 부모요소가 relative일떄,
    그 부모요소를 기준으로 정한 수치만큼 떨어져 배치
    (부모요소가 없다면 body가 기준)
This div element has position: relative;
This div element has position: absolute;
