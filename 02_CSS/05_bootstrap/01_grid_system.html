부트스트랩 그리드시스템의 이해

  • 컨테이너 (.container)
  • 행 (.row)
  • 열 (.col-*-*)
행은 반드시 컨테이너 클래스 안에 넣어야 함(중요!!)
기본적으로 행은 컨테이너 클래스의 레이어 및 레이아웃의 영향을 받게 설계되어있음.
열은 반드시 행 클래스 안에 넣어야 함(중요!)
오직 열만 행의 직계자손이 될 수 있음.
컨텐츠는 꼭 열 클래스 안에 넣어야 함.
이 규칙을 지키면 예상하는 레이아웃이 구성됨.
