색상(color)

텍스트 색상은 오직 color만 사용

이외에는 색상을 설정하려고 하는 부분의 접두사를 붙임

  • 배경색 : background-color: ??;
  • 경계선색 : border-color: ??;

색을 HEX로 표현할 때 단축형으로 사용 가능.

일반적으로 HEX는 6자리의 수나 문자로 이루어져 있다 #[R(2)G(2)B(2)]. 이 때 각 색상을 나타내는 두자리의 숫자나 문자가 같다면, 한 자리의 수나 문자로 통일할 수 있다. 이 때의 단축형은 항상 3자리의 문자나 숫자로 이루어져 있어야 한다.
ex) #22ff99 => #2f9
